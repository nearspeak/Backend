README

This is the Nearspeak Backend Documentation.

Required Software:

  • Ruby version: 2.2.2

  • Rails version: 4.2.0

Installation:

  • Config your database config/database.yml

  • Config your production environment config/environments/production.rb

  • Config your nearspeak server config config/initializers/nearspeak_server_config.rb

  • Config your secret token config/initializers/secret_token.rb

Optional:

  • Config your social network keys config/social_keys.yml
